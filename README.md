# Chatbot
AI Based Sanchar

### Requirements
    Python = 2.x.x
    Flask
    Aiml
    pip

## Installation

1. Install the required packages.
    pip install flask
    pip install aiml

2. Run the python server.
   main.py
   
3. Open "http://127.0.0.1:5000" in your browser.

4. You're done and let's chat with your Robot via browser.
